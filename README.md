## Installation

From PyPI distribution system:

pip install Zeras

</br>

Or from the code:

git clone https://github.com/Li-Ming-Fan/Zeras

cd Zeras

make clean install

</br>

Or just incorporate the diractory Zeras/Zeras/ to you project.

</br>
